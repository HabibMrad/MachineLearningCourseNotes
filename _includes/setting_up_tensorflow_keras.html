Setting up Tensorflow and Keras

In terminal, run 

python3 -m pip install tensorflow
python3 -m pip install keras
As of 11/21/2019, Tensorflow doesn't officially support python 3.8 yet, so if you have that, you'll need to downgrade to Python 3.7.6.
First, get the python 3.7.6 installer here. Run the installer.
Open a new Terminal window. To confirm you're now running python 3.7.6, run "python --version" and see that it displays 3.7.6.
Install tensorflow and keras as above: 
python3 -m pip install tensorflow
python3 -m pip install keras
You'll also need to reinstall previous packages, to make them run in Python 3.7.6: 
python3 -m pip install matplotlib
python3 -m pip install scipy
python3 -m pip install scikit-learn
python3 -m pip install jupyter

Troubleshooting

If you get an error like 

ModuleNotFoundError: No module named 'theano'
then you need to switch the backend to tensorflow. See the instructions at https://keras.io/backend/ or ask for help.
