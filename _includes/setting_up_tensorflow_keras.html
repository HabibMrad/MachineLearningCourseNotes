Setting up Tensorflow and Keras

Unfortunately, Tensorflow does not currently support the latest version of Python (3.7), so for now you will need to get an earlier Python version and install your libraries there.

  1. Download Python 3.6 for 64-bit OSX and run this installer.
  2. There should now be a folder called "Python 3.6" in your Applications folder. Go to that folder in Finder and double click "Install Certificates.command" to install the SSL certificates that will allow you to download datasets later.
  3. In terminal, run 
    python3 --version
    and make sure you get the output 
    Python 3.6.7
    A If you don't, please let me know. We'll need to reset the links.
  4. In terminal run 
    pip3.6 install --upgrade tensorflow
  5. Usually, we are not going to be using Tensorflow itself, but some other higher-level library like Keras. 
    pip3.6 install --upgrade keras

Note that after you finish all of this python3 will no longer run the Python in which you installed scikit-learn, so you may want to reinstall scikit-learn using pip3.6. That other, newer version of Python is still on your system, and you can use it by putting the version in the command: 

python3.7
