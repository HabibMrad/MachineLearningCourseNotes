Setting up Tensorflow and Keras

In terminal, run 

pip3 install tensorflow
pip3 install keras
As of 11/21/2019, Tensorflow doesn't officially support python 3.7 yet, so if you have that, you'll need to downgrade to Python 3.6.
First, get the python 3.6 installer here.
Open a new Terminal window. To confirm you're now running python 3.6, run "python --version" and see that it displays 3.6.
Then, install tensorflow and keras as above: 
pip3 install tensorflow
pip3 install keras
You'll also need to reinstall previous packages, to make them run in Python 3.6: 
pip3 install matplotlib
pip3 install scipy
pip3 install scikit-learn
pip3 install jupyter
